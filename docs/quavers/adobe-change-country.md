# 记录一下 Adobe 账号转区

我的 Adobe 账号是大一时注册的美区号，来了日本之后也没管，就一直这么用着。直到最近用的日区 Amex 卡出了满 1600 减 400 的福利，再加上高市早苗猛搞百亿补贴，就想把账号切到日区。

一开始我看的是 [Adobe 官方的这个帮助页面](https://helpx.adobe.com/jp/x-productkb/policy-pricing/change-country-associated-with-adobe-id.html)，退订了当前的美区订阅，等待订阅过期之后绑新卡重新订。结果过期之后开新订阅，选择支付方式的时候发现自己被迫拥有了钱学森同款待遇，账号被锁在美区出不了国了。

![=360x](https://i.see.you/2026/04/07/Eon7/1Capture_2026-04-07_122835.webp)

于是我找客服对线，提出要求换地区之后他捣鼓了半天，来了一句不好意思你当前的账号换不了，但是可以重新注册一个日区账号。

![=360x](https://i.see.you/2026/04/07/ibV6/20260407115513_8895_2.webp)

我就怒了，说你告诉我不能换的理由，我已经用了你们家公司服务八年了不少数据都在 creative cloud 上面，你知道数据迁移起来有多麻烦吗？

> 客服：输入中
> 客服：您好，您的账号可以换，请您提供一下现在的日本地址。
> 我：翻斗大街翻斗花园2号楼1001室
> 客服：给您换完啦

![=360x](https://i.see.you/2026/04/07/Boi6/1Capture_2026-04-07_123412.webp)

所以我是韭菜认证通过了是吗？
